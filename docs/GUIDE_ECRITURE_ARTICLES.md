# Guide d'utilisation – Écrire un article

Ce guide explique comment créer et formater vos articles dans l'espace membre (Dashboard).

---

## Accéder à l'éditeur

1. Connectez-vous à votre compte sur le site
2. Allez dans **Mon espace** (ou **Dashboard**)
3. Cliquez sur **Nouvel article** dans la section « Mes articles »

L'éditeur s'ouvre en plein écran pour vous laisser écrire à l'aise.

---

## Structure de l'article

### 1. Titre
Saisissez le titre dans le champ principal en haut. Le **slug** (URL) est généré automatiquement à partir du titre.

### 2. Résumé
Court texte affiché dans la liste des articles. Indiquez en une ou deux phrases de quoi parle l'article.

### 3. Image de couverture
- Cliquez sur **« Image de couverture »** (ou **« Changer la couverture »** si une image existe déjà)
- Choisissez une ou plusieurs images
- Cliquez sur une vignette pour la définir comme couverture
- Cliquez sur **« Insérer »** sous une vignette pour l’ajouter dans le corps de l’article

### 4. Contenu

L’article est composé de **blocs** : paragraphes de texte et images.

#### Ajouter du contenu
- **Paragraphe** : ajoute un bloc de texte
- **Image** : ajoute un bloc image (ou ouvre le sélecteur de fichier pour un bloc image vide)

#### Organiser les blocs
Chaque bloc a des boutons à gauche :
- **↑** : remonter le bloc
- **↓** : descendre le bloc  
- **🗑️** : supprimer le bloc

---

## Mise en forme du texte

Dans vos paragraphes, vous pouvez utiliser des **codes simples** pour la mise en forme :

| Code           | Effet           | Exemple                  |
|----------------|-----------------|---------------------------|
| `**texte**`    | **Gras**        | `**mot important**`       |
| `*texte*`      | *Italique*      | `*citation*`              |
| `[1]` `[2]`    | Référence       | Renvoi vers les sources   |
| Retour à la ligne | Nouveau paragraphe | Normal en tapant Entrée |

### Exemples

```
Selon une étude récente **publiée en 2024** [1], le marché de l'art *continue de croître*.

Cette analyse souligne l'importance des *ventes aux enchères* [2] dans la valorisation des œuvres.
```

---

## Références et sources

### Insérer une référence dans le texte
- Tapez `[1]`, `[2]`, `[3]`, etc. à l’endroit voulu
- Ou cliquez sur le bouton **« Réf »** dans un bloc texte pour insérer le numéro suivant

### Renseigner les sources
En bas de l’éditeur, dans la section **Sources** :
1. Cliquez sur **« Ajouter une source »**
2. Renseignez le texte pour chaque numéro (ex. : auteur, titre, éditeur, année, URL)

Exemple de source :  
`Dupont J., L'Art contemporain en France, Éditions XYZ, 2024.`

Les références `[1]`, `[2]` dans le texte seront automatiquement transformées en liens vers les sources affichées en bas de l’article.

---

## Paramètres

En haut à droite de l’éditeur :
- **Rubrique** : catégorie de l’article (Histoire des arts, Marché de l'art, etc.)
- **Statut** : Brouillon ou Publié
- **À la une** : afficher ou non l’article en mise en avant
- **Enregistrer** : enregistre les modifications

---

## Bonnes pratiques

1. **Enregistrer régulièrement** pour ne pas perdre vos modifications
2. **Prévisualiser** avant de publier (ouvrir l’article en statut Brouillon)
3. **Rédiger un résumé court** (2–3 lignes) pour la liste des articles
4. **Numéroter les références** dans l’ordre d’apparition dans le texte
5. **Formater les sources** de manière homogène (ex. : Auteur, Titre, Éditeur, Année)

---

## Raccourcis de mise en forme

| Touche / action | Résultat                |
|-----------------|-------------------------|
| `**` + mot + `**` | Texte en **gras**    |
| `*` + mot + `*`   | Texte en *italique*  |
| `[1]`             | Référence numéro 1   |
| Bouton **Réf**    | Insère la référence suivante |
