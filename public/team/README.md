# Photos de l'Équipe

Placez ici les photos des membres de l'équipe :

- `president.jpg` - Photo du président
- `vice-president.jpg` - Photo du vice-président
- `redactrice.jpg` - Photo de la rédactrice en chef
- `secretaire.jpg` - Photo du secrétaire
- `dev.jpg` - Photo du développeur web

## Format Recommandé
- Format : JPG ou PNG
- Taille : 500x500 pixels minimum
- Poids : < 500 Ko (optimisé)
- Style : Photo professionnelle, fond neutre de préférence
