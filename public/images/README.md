# Images du Site

## Structure
- `articles/` - Images pour les articles
- `rubriques/` - Images pour les rubriques
- `decoratives/` - Images décoratives pour le site

## Conseils
- Optimisez vos images avant de les uploader
- Formats recommandés : JPG, PNG, WebP
- Utilisez des noms descriptifs
- Compressez les images (outils : TinyPNG, ImageOptim)
